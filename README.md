# 🍽 MZBR

> **MZBR**은 숏폼으로 맛집 리뷰를 제공하는 웹 서비스 입니다
> 

**기간** : 2023.10.10 ~ 2023.11.17


## 💡 **주요 기능**

> **MZBR**은 식당에 대한 리뷰 영상을 등록하고 시청하는 데에 편리하도록 다양한 기능을 제공합니다.
